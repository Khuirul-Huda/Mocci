Mocci bot, Discord bot used in my discord server

# Installation
## Requirements
* NodeJS 17.x
* NPM

## Steps
1. Clone this repository
2. Go to folder
```bash
cd Mocci
```
3. Install neccesary packages
```bash
npm i
```
4. Rename sample.env to .env
5. Edit .env file
```
BOT_TOKEN=BOT TOKEN HERE
TEST_GUILD_ID=TEST GUILD ID
```
6. Start discord bot
```bash
npm run deploy
```
